Printing/Output

Printing is usually done with 'echo'. We Can echo a Number, String, Variable anything we would like. When echoing a variable we always put a '$' sign before it. For printing arrays we do ${arrayname[indexID]}. It is always best to quote the variables which means putting them in "quotation marks". Here is a script of printing different variables.

You can also directly print a terminal command, the syntax followed is ' echo "$(command)" '. Here is an example of a script that prints the addition of 2 number in 1 line!
